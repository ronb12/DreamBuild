### Prerequisites
- Node.js 18+ 
- npm or yarn
- Firebase CLI

### Setup
```bash
# Clone the repository
git clone https://github.com/ronb12/DreamBuild.git
cd DreamBuild

# Install dependencies
npm install

# Install Firebase CLI
npm install -g firebase-tools

# Login to Firebase
firebase login

# Initialize Firebase
firebase init

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to Firebase
firebase deploy
```
